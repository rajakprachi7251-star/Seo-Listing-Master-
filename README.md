## ✨ Features

- 🤖 **AI-Powered Generation** - Uses OpenAI GPT-3.5 for intelligent content creation
- 🛒 **Multi-Marketplace Support** - Amazon, Walmart, Myntra, Flipkart
- 📊 **Real-time SEO Scoring** - Calculates optimization score (0-100 points)
- 🎯 **Optimized Titles** - Marketplace-specific character limits
- 📌 **5 Bullet Points** - Compelling, keyword-rich product benefits
- 📝 **Product Descriptions** - Detailed, SEO-optimized content
- 🔑 **Backend Keywords** - 5 custom keyword sets for each product
- 💾 **Easy Export** - Download listings as text files
- 🎨 **Beautiful UI** - Streamlit-based responsive interface

- **Frontend:** Streamlit 1.28.1
- **Backend:** Python 3.8+
- **AI:** OpenAI GPT-3.5
- **Version Control:** Git & GitHub
- **Deployment:** Streamlit Cloud

## 📦 Installation

### Local Development

```bash
# Clone the repository
git clone https://github.com/rajakprachi7251-star/Seo-Listing-Master-.git
cd Seo-Listing-Master-

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Create .env file
echo OPENAI_API_KEY=your_api_key_here > .env

# Run the app
streamlit run app.py
```

The app will open at `http://localhost:8501`

## 💻 How It Works

1. **Input Product Details**
   - Product name
   - Category
   - Keywords
   - Optional: Existing description

2. **Customize Settings**
   - Select marketplace (Amazon, Walmart, Myntra, Flipkart)
   - Choose tone (Professional, Casual, Luxury, Budget-Friendly, Technical)
   - Set target audience

3. **Generate Content**
   - AI generates optimized title
   - Creates 5 compelling bullet points
   - Writes detailed product description
   - Generates backend keywords

4. **Analyze & Export**
   - View SEO optimization score
   - See content statistics
   - Download as text file
   - Copy-paste to marketplace
